# Accessibility status

The desktop application targets keyboard and platform accessibility-service
interoperability. Main navigation, the map, spectrum/waterfall, tuning wheel and
scanner selections have accessible names or help. Map and tuning functions have
keyboard equivalents.

This is not yet a Section 508 conformance claim. Before procurement, the exact
release must receive manual keyboard, focus-order, scaling, contrast and
screen-reader testing on each target platform. Results should be documented in
an Accessibility Conformance Report (ACR/VPAT) using the applicable Revised 508
software criteria and WCAG 2.0 Level A/AA requirements.
